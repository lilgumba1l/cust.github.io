Выберите роль

Общие команды

  • /news — Узнать все новые новости
  • /msg [ник] [сообщение] — Отправить личное сообщение игроку
  • /spawn — Телепортироваться на спавн
  • /changepass [ваш пароль] [новый пароль] — Смена пароля

Команды для ролей

Президент

  • /lp user [ник] parent add [названия роли] — Добавить роль игроку
  • /lp user [ник] parent remove [названия роли] — Удалить роль игроку
  • /rg info — Узнать информацию об регионе и изменить её
  • Создавания роли — Для создания роли вы должны написать Тех. Админу в Discord aris.now и у роли должно быть определённое предназначения.

Журналист

  • /broadcast [сообщение] — Отправить сообщение, которое уведёт весь сервер.
  • Создания газет — Для создания газеты вы должны написать Тех. Админу в Discord aris.now и написать что будет в газете, и газета появиться на следующий день.

Команды для привата

  • /rg claim [название] — Создать приватную территорию
  • /rg addmember [название] [ник] — Добавить игрока в приват
  • /rg removemember [название] [ник] — Удалить игрока из привата
  • /rg addowner [название] [ник] — Добавить владельца в приват
  • /rg removeowner [название] [ник] — Удалить владельца из привата
  • /rg remove [название] — Полностью удаляет регион
  • /rg info — Узнать информацию об регионе
